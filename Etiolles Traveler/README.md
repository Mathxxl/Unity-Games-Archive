# Étiolles' Traveler

## Overview

Étiolles' Traveler est un jeu de survival action où le joueur doit se battre avec différentes armes contre la faune locale et commercer avec les Magdaléniens pour améliorer son équipement, le tout sous fond de références historiques précises.
